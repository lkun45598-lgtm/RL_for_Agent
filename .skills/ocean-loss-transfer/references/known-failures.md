# 已知失败模式（基于 71 次实验）

本文档记录已被证实会导致训练失败的 loss 组件和配置。

---

## 硬性拦截（Critical）

这些模式会导致训练完全崩溃，**必须避免**。

### 1. SSIM Loss
- **失败实验**: exp#11
- **结果**: SSIM=0.109（所有模型崩溃）
- **原因**: SSIM 作为 loss 导致梯度不稳定
- **建议**: 使用 SSIM 作为评估指标，不要作为 loss

### 2. Laplacian
- **失败实验**: exp#20, exp#40, exp#66
- **结果**: SSIM=0.439 / 0.6349（严重下降）
- **原因**: 二阶导数放大噪声
- **建议**: 使用一阶梯度（Sobel）

### 3. 5x5 Sobel
- **失败实验**: exp#38
- **结果**: SSIM=0.6395
- **原因**: 梯度过度平滑
- **建议**: 使用 3x3 Sobel

### 4. Relative FFT（除法）
- **失败实验**: exp#36
- **结果**: SSIM=0.088（崩溃）
- **原因**: 频域除法数值不稳定
- **建议**: 使用残差 FFT（`rfft2(pred - target)`）

### 5. Scale-8 多尺度
- **失败实验**: exp#37
- **结果**: SSIM=0.6156
- **原因**: 信息损失过大
- **建议**: 最大 scale 不超过 4

---

## 风险警告（Warning）

这些模式在某些情况下可能失败，需谨慎使用。

### 1. Charbonnier Loss
- **风险**: FNO2d 模型敏感
- **建议**: 先在 SwinIR 上测试

### 2. Wavelet Transform
- **风险**: 计算开销大
- **建议**: 确保 GPU 显存充足

### 3. Power Spectrum
- **失败实验**: exp#53
- **风险**: 数值不稳定
- **建议**: 使用幅度谱（amplitude）

---

## 安全组件

这些组件已被验证安全有效：

- **Multi-scale [1,2,4]**: exp#13 baseline
- **Residual FFT**: exp#41 最优
- **3x3 Sobel**: 稳定
- **Relative L2**: 稳定
